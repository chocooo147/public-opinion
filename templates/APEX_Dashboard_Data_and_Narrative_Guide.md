# APEX 综合看板数据口径与叙述规则

## 1. 适用范围

- 报告范围：中国地区，2026-W25 至 2026-W29。
- W29 自然周：2026-07-13 至 2026-07-19（Asia/Shanghai）。
- W29 数据类型：有限真实样本与模型输出，`formal_reporting_qualified=false`。
- B站单位为映射后的评论样本；小黑盒单位为登录态公开搜索结果中的可见帖子。小黑盒未采集评论正文。

## 2. 综合看板口径

- “综合”只表示把不同平台的有限观察放在同一主题框架中审阅，不表示平台全量。
- B站评论与小黑盒可见帖子单位不同，不得直接相加后解释为“平台总声量”；界面中的混合观察数只能表述为“混合文本观察”。
- 平台趋势必须分平台展示。小黑盒搜索样本扩大不能解释为平台讨论量增长。
- SnowNLP 情感、负面率和风险分均为模型结果，未经独立人工校准；低置信度、离群和可能误分项需人工复核。
- 缺失的官方观看、直播或互动指标不得以讨论量替代。

## 3. 叙述顺序

每条结论按以下顺序书写：

1. 结论：本周发生了什么。
2. 驱动：由哪些主题或事件推动。
3. 证据：给出样本量、变化、风险分和代表文本。
4. 对比：只与口径一致的历史周或同平台数据比较。
5. 影响：说明对产品、社区或传播工作的含义。

## 4. 禁止性规则

- 不把有限公开搜索样本描述为平台普查或全量采集。
- 不把 Mixed 强制改写为 Neutral。
- 不把相关性写成因果关系；未经验证的因果必须标记为假设。
- 不跨平台直接比较或相加定义不同的指标。
- 不用最终主题注册表的累计数据回填历史周；历史链必须截断在所选周。
- 每个主题最多展示三条代表文本，并注明其只代表样本观点。

## 5. 建议发布说明

> 本报告基于有限真实样本与模型输出，仅用于探索和人工复核，不具备正式统计报告资格。B站评论与小黑盒公开搜索可见帖子单位不同，综合视图不代表平台总量。
